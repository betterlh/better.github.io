实现多线程有两种方法，第一个方法是使用继承thread，使用thread子类来创建线程，第二种是使用runnable接口。
第一个比较简单，其实thread也是有接口的，通过查thread类可以发现，在最前面有

说明他也是用到Runnable接口的.thread的子类重写了run方法，然后thread又是用到Runnable接口。这才实现了线程。
